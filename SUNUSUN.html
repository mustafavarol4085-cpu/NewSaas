Sistem Yetenekleri - Rep vs Manager

📱 Rep (Satış Temsilcisi)

  • ✓ Çağrı geçmişini görmek
  • ✓ Puanları ve feedback'i almak
  • ✓ Ses ve transkripti dinlemek
  • ✓ AI Coach'a soru sormak
  • ✓ Gelecek çağrıları planlamak
  • ✓ Kişisel beceri izlemesi görmek

📊 Manager (Yönetici)

  • ✓ Takım performansını görmek
  • ✓ Temsilcileri kar/zarar ile karşılaştırmak
  • ✓ Trend analizi yapmak
  • ✓ Bireysel drilldown yapmak
  • ✓ Takım için AI insights almak
  • ✓ Zamanlanmış çağrıları yönetmek

💡 Sonraki Adımlar:

✓ Müşteri bilgi temeli entegre edilince → Playbook referansları yapabiliyor

✓ Chorus entegre edilince → Gerçek çağrı verileriyle çalışabiliyor

✓ Canlı panel eklenince → Aktif çağrılar sırasında danışma alabilir
